Environment Variables Unit
ReadMe

Description

This unit provides a static class, a component and a set of routines for managing and manipulating environment variables. Full details can be found in the following locations:

  • Environment Variables Unit Home Page.
  • The project's online documentation.

In addition there is an article, How to access environment variables, that discusses interaction with environment variables on Windows.

The unit is believed to be compatible with all Win32 editions of Delphi from Delphi 4 onwards. It has been tested with Delphi 4, 6 and 7, 2006 and 2010.

Installation

The Environment Variables Unit and documentation are supplied in a zip file. Before installing you need to extract all the files from the zip file. The following files will be extracted:

  • PJEnvVars.pas – component and routine source code.
  • PJEnvVars.dcr – component palette glyph.
  • ReadMe.htm – this read-me file.
  • DemoCode.htm – examples of how to use the Environment Variables Unit.
  • ChangeLog.txt – the project change log.
  • MPL-2.txt – the Mozilla Public License v2.0.
  • Documentation.url – short-cut to the unit's online documentation.

You can now install the component into the Delphi IDE. To do this, the files PJEnvVars.pas and PJEnvVars.dcr should be added to a design time package. If you need help doing this see here.

Should you just want to use the routines, or create the component dynamically at run time you don't need to install the unit into the IDE and can simply either places it somewhere on your library path or include it in your projects.
